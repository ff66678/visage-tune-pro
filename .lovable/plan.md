

## Plan: 课程库布局多样化，打破单调

### 问题
当前每个分类都是「一个大卡片 + 2列网格」的统一模板，视觉上太重复单调。

### 改动文件
**`src/components/LibraryPage.tsx`**

### 设计方案 — 交替使用 3 种分类布局

根据分类的索引（idx % 3），交替使用不同的展示方式：

**布局 A（idx % 3 === 0）— 当前样式**
- 第一个课程做大横幅卡片（160px），其余 2 列网格
- 保持不变

**布局 B（idx % 3 === 1）— 横向滚动列表**
- 所有课程排成一行横向滚动，每个卡片宽度 `w-[200px]`
- 图片更高（130px），底部叠加文字信息
- 类似"热门排行"的横滑体验，但卡片更大

**布局 C（idx % 3 === 2）— 单列竖排列表**
- 每个课程做成左图右文字的横向卡片（类似播放列表样式）
- 左侧图片 `w-24 h-20 rounded-xl`，右侧标题 + 时长 + 评分 + 难度标签
- 紧凑信息密度，视觉节奏不同

### 不变的部分
- 数据源、筛选、排序、路由跳转、收藏逻辑全部不变
- 顶部 Banner 和热门排行不变
- 只改分类组内课程卡片的排列方式
